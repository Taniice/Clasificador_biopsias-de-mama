Clasificador de Biopsias de Mama usando el Perceptrón

Este proyecto implementa un clasificador binario utilizando la arquitectura de un Perceptrón, desarrollado en Python con apoyo de NumPy y Scikit-learn.

Como caso de estudio se utilizó el conjunto de datos Breast Cancer Wisconsin, 
el conjunto de datos contiene características extraídas de imágenes digitalizadas de biopsias de mama, utilizadas para clasificar cada muestra en una de dos clases (Tumor benigno y Tumor maligno)

Tecnologías utilizadas:
-Python
-Google Colab
-NumPy
-Pandas
-Scikit-learn
-Matplotlib

Modelo utilizado:
Se implementó un Perceptrón, uno de los algoritmos más representativos de las redes neuronales artificiales, diseñado para resolver problemas de clasificación binaria.
El modelo aprende ajustando sus pesos durante el entrenamiento hasta encontrar un hiperplano que permita separar las dos clases.

Cómo ejecutar el proyecto: 
Clonar este repositorio.
Abrir el notebook en Google Colab o Jupyter Notebook.
Instalar las dependencias necesarias.
Ejecutar las celdas en orden.
